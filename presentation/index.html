Angular

  • SPA Framework
  • Тяжело внедрить

Angular Elements

  • Поддержка браузерами
  • Размер 200-300 Kb
  • Сложность
  • Ivy должен помочь

Vue

  • Простой в освоении
  • Размер 21 Kb
  • Различные варианты интеграции
  • new Vue({}) для каждого компонента
  • Нужно писать кастомные фабрики для vuex

Vue Custom Element

  • Решает проблему new Vue({})
  • Native API или Fallback
  • Lazy loading
  • vue-custom-element-vuex
  • Компоненты необходимо добавлять в Vue.config.ignoredElements

React

  • Лучшие подходы
  • Размер в 30 Kb
  • Вызов ReactDOM.render для каждого компонента
  • Оборачивать каждый компонент в Redux Provider

React Habitat

  • Маленький размер 4 Kb
  • Code-splitting
  • Возможность создавать кастомные фабрики
  • react-habitat-redux
  • Proxy
  • Различные типы props

Preact Habitat

  • Маленький размер 4 Kb
  • Идеальный старт для создания виджетов
  • Самый простой API
  • Нет кастомных фабрик
